# Entity Documentation - minecraft:persistent

`minecraft:persistent` defines whether an entity should be persistent in the game world.

## Example

```json
"minecraft:persistent":{
}
```

## Vanilla entities examples

### armor_stand

```json
"minecraft:persistent": {
}
```
