# Entity Documentation -  minecraft:is_shaking

`minecraft:is_shaking` sets an entity is currently shaking.

## Example

```json
"minecraft:is_shaking":{
}
```

## Vanilla entities examples

### piglin

```json
"minecraft:is_shaking":{
}
```
