# Entity Documentation - minecraft:item_hopper

`minecraft:item_hopper` allows an entity to function like a `hopper` block.

## Example

```json
"minecraft:item_hopper":{
}
```

## Vanilla entities examples

### hopper_minecart

```json

        }
```
