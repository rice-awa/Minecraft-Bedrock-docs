# Entity Documentation - minecraft:hide

`minecraft:hide` compels an entity to move to and hide at their owned POI or the closest nearby.

## Example

```json
  "minecraft:hide": {}
```

## Vanilla entities examples

### villager_v2

```json
"minecraft:hide": {}
```
