# Item Documentation - minecraft:tags

`minecraft:tags` determines which tags are included on a given item.

## Parameters

|Name |Default Value  |Type  |Description  |
|:----------|:----------|:----------|:----------|
| tags|*not set* | Array| An array which can contain multiple item tags. |
